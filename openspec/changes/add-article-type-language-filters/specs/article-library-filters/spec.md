## ADDED Requirements

### Requirement: 按文章类型和语言筛选文章库

系统必须（MUST）允许运营人员在文章库中按文章类型、语言或两者组合筛选文章，并与已有的关键词和状态筛选共同生效。

#### Scenario: 按单个条件筛选
- **当** 运营人员选择一种文章类型或一种语言并执行查询
- **则** 系统只展示满足已选条件的文章

#### Scenario: 按多个条件组合筛选
- **当** 运营人员同时选择关键词、状态、文章类型和语言中的多个条件并执行查询
- **则** 系统只展示同时满足所有已选条件的文章

#### Scenario: 重置筛选条件
- **当** 运营人员点击重置
- **则** 系统清空关键词、状态、文章类型和语言，并重新展示第一页文章

#### Scenario: 保留分页筛选条件
- **当** 运营人员在已筛选的文章列表中切换分页
- **则** 系统使用当前筛选条件加载目标页文章
